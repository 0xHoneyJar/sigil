# Sigil Evidence

This directory contains evidence supporting design decisions.

## Purpose

When patterns are validated through usage or user research, evidence is stored here:
- Usage metrics supporting a pattern
- User feedback on interactions
- A/B test results
- Accessibility audit findings

## Format

Evidence files can be:
- Markdown summaries
- JSON metrics
- Screenshots (in subdirectories)
